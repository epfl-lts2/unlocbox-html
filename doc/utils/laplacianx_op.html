LAPLACIANX_OP - dimentional Laplacian

Usage

[Lx] = laplacianx_op( I );

Input parameters

I Input image

Output parameters

Lx Laplacian along x

Description

Compute the sum of the laplacian along x. This operator is self-adjoint.

\begin{equation*} \mathcal{L}_x = I_{xx} \end{equation*}
