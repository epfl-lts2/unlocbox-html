DIV_OP3D - Divergence operator in 3 dimentions

Usage

I = div_op3d(dx, dy, dz)
I = div_op3d(dx, dy, dz, wx, wy, wz)

Input parameters

dx Gradient along x
dy Gradient along y
dz Gradient along z
wx Weights along x
wy Weights along y
wz Weights along z

Output parameters

I Output image

Description

Compute the 3-dimentional divergence of a 3D-image. If a 4 dimentional signal is given, it will compute the divergence of all cubes in the 4 diementionals signal.

Warning this function compute the divergence operator defined as minus the adjoint of the gradient

\begin{equation*} \text{div} = - \nabla^* \end{equation*}
