Installation

To install, simply unpack the package. The toolbox is contained in the 'unlocbox' directory and in all the subdirectories. To use the toolbox type init_unlocbox in the 'unlocbox' directory. This will set up the necessary paths.

Documentation is available online or in the note section. See the userguide , the full documentation and a tutorial.
